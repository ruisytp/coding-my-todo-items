# YTP代办事项网站开发指令

## 项目概述
使用 Vue3 + TypeScript + Tailwind CSS + Vite 开发的个人作品集网站。

## 开发规范
- 使用函数式组件
- 使用 Tailwind CSS 编写样式
- 组件要可复用
- 代码要有注释

## 设计要求
- 浅色主题
- 使用渐变色作为强调色
- 添加平滑的滚动动画
- 确保移动端适配

## 代码风格

- **风格规范**：
  - 采用 ESLint + Prettier 统一代码风格
  - 2 空格缩进，单引号，分号可选
  - 组件/文件命名采用大驼峰（PascalCase），变量/函数小驼峰（camelCase）
  - 类型、接口以大写字母开头
- **注释规范**：
  - 关键逻辑、接口、类型需有中英文注释
  - 复杂流程、SSE 事件、RAG 状态等需详细说明
- **样式规范**：
  - 全局样式集中于 `style.css`，组件样式优先使用 TailwindCSS
  - 避免内联样式，复用 Tailwind 工具类

## 注意事项
- 保持设计简洁，不要过度设计
- 性能优化：图片使用懒加载
